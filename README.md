# 🏝️ Survivor

[cite_start]**Survivor** est un jeu de survie développé en **C#**, représentant mon premier projet d'envergure en programmation orientée objet[cite: 22, 47]. Plongé dans un environnement hostile, le joueur doit faire preuve de stratégie et de gestion de ressources pour rester en vie.

## 🌟 Pitch du Projet

[cite_start]Échoué sur une île déserte, votre objectif est simple mais vital : **survivre pendant 10 jours** jusqu'à l'arrivée des secours[cite: 47]. [cite_start]Chaque journée est un défi où vous devrez gérer votre faim, votre soif et votre énergie tout en composant avec le tempérament imprévisible des dieux de la mythologie grecque[cite: 47].



## 🚀 Fonctionnalités Principales

* **Gestion des Besoins Vitaux** : Le joueur doit surveiller son énergie. [cite_start]Pour regagner des forces, il est impératif de se désaltérer dans les rivières et de se nourrir chaque jour[cite: 47].
* [cite_start]**Écosystème Dynamique** : De la nourriture et des animaux apparaissent de manière aléatoire sur la carte chaque jour, obligeant le joueur à explorer constamment son environnement[cite: 47].
* **Système d'Offrandes Divines** : Un mécanisme de jeu unique basé sur la mythologie grecque. [cite_start]Le joueur doit faire des offrandes régulières aux dieux sous peine de subir des malédictions impactant sa survie[cite: 47].
* [cite_start]**Inventaire et Gestion d'Objets** : Le joueur peut ramasser, stocker ou jeter des objets[cite: 47]. [cite_start]Attention : les denrées sont périssables et peuvent devenir périmées si elles ne sont pas consommées à temps[cite: 47].
* [cite_start]**Cycle Sommeil/Activité** : La survie passe aussi par le repos pour restaurer l'énergie nécessaire aux expéditions du lendemain[cite: 47].

## 🎮 Commandes du Jeu

[cite_start]Le jeu se pilote entièrement au clavier pour une expérience immersive[cite: 47]:

| Touche | Action |
| :--- | :--- |
| **W, A, S, D** | [cite_start]Déplacements sur l'île [cite: 47] |
| **I** | [cite_start]Interaction (Boire, Manger, Tuer un animal) [cite: 47] |
| **Z** | [cite_start]Gestion d'objet (Ramasser / Laisser par terre) [cite: 47] |
| **N** | [cite_start]Dormir / Passer au jour suivant [cite: 47] |
| **Q** | [cite_start]Quitter le jeu [cite: 47] |

## 💻 Technologies Utilisées

* [cite_start]**Langage** : C# [cite: 31, 47]
* [cite_start]**IDE** : Rider [cite: 35, 47]
* [cite_start]**Paradigme** : Programmation Orientée Objet (POO) pour la gestion des entités (joueur, animaux, objets, dieux)[cite: 47].

## 🛠️ Structure Technique

Le projet met l'accent sur :
1.  [cite_start]**L'aléatoire** : Algorithme d'apparition (spawn) quotidien pour les ressources et les animaux[cite: 47].
2.  [cite_start]**La gestion du temps** : Système de péremption des objets basé sur le cycle des jours[cite: 47].
3.  [cite_start]**La logique de collision** : Interactions avec les éléments de la carte comme les rivières[cite: 47].
